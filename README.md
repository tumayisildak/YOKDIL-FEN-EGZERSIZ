# YOKDIL-FEN-EGZERSIZ

Bu proje, YÖKDİL Fen Bilimleri sınavına hazırlanan öğrencilerin kelime dağarcığını geliştirmek amacıyla tasarlanmış web tabanlı bir öğrenme aracıdır. 54 gün boyunca her güne özel seçilmiş akademik kelimeleri interaktif bir akışla öğretir.

### Özellikler
- **4 Aşamalı Öğrenme:** Öğrenme, Eş Anlamlı Eşleştirme, Türkçe Karşılık Eşleştirme ve Test aşamaları.
- **Sesli Okuma (Text-to-Speech):** Kelimelerin doğru telaffuzlarını dinleme imkanı.
- **Gelişmiş Navigasyon:** Öğrenme modunda kelimeler arasında ileri-geri gidebilme.
- **Progress Tracking:** Tamamlanan günlerin yerel depolama (localStorage) ile kaydedilmesi.
- **Responsive Tasarım:** Modern ve karanlık tema odaklı kullanıcı arayüzü.

### Teknolojiler
- HTML5
- CSS3 (Custom Variables & Modern UI Elements)
- Vanilla JavaScript (Web Speech API)
